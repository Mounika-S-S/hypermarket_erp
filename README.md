# Hypermarket ERP

Offline-first ERP application built using Flutter.

## Features
- Product management
- Inventory tracking (Stock IN / OUT)
- Sales entry with automatic stock deduction
- Local SQLite database
- Works completely offline

## Tech Stack
- Flutter
- Dart
- SQLite (sqflite)

## Roadmap
- v1.0: Offline ERP MVP
- v2.0: Cloud sync & SaaS (planned)
